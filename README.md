# ARDL Model for Behavioral Finance: Implications of Herd Behavior in the S&P 500

This project implements an **Autoregressive Distributed Lag (ARDL)** model to evaluate whether herding behavior, proxied by news sentiment and market volatility (VIX), helps explain short‑run fluctuations in daily S&P 500 returns.
The analysis is based on a daily dataset spanning 2015–2025 and includes five core variables:

Dependent variable:

* S&P 500 returns

Explanatory variables:

* News Sentiment Index

* VIX Index

* Gold returns

* Federal funds effective rate

This project implements the empirical analysis developed in my master’s thesis, focusing on ARDL modeling of behavioral market drivers.
